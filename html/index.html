Программа работы с двумерным динамическим массивом Документация

Программа предназначена для работы с двумерным динамическим массивом. Функции:

  • Ввод размера массива;
  • Ввод элементов массива:
  1. Вручную;
  2. С помощью генератора случайных чисел;
  • Обработка массива:
  1. Поиск максимального и минимального элементов массива
  2. Обнуление строки с наименьшим элементом
  3. Удаление столбца с максимальным элементом
  • Вывод массива на экран.
